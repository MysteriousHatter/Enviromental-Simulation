    How to add grass

    Steps

    1. Add "Plant Growth Phase Manager" component to a GameObject

    2. Create a "Grass Growth Phase" asset

    3. Configure the asset

    4. Add "Grass Growth Terrain Manager" component in the same place as the "Terrain" component

    5. Insert the desired phase in the field of the component and click on the "Add" button

    6. Select "Paint Details" on the "Terrain" component and use the new prototype added to the terrain through the "Scene" window
